# Todo-Listen-Projekt

Basisprojekt fuer eine Todo-Liste auf React/Vite-Basis.  
Ziel ist ein sauberes, wartbares Grundgeruest als Ausgangspunkt fuer weitere Features.

## Tech-Stack

- React
- Vite
- ESLint
- npm

## Installation

```bash
npm install
```

## Projekt starten

```bash
npm run dev
```

Die Startseite zeigt den Titel **Todo Liste** und eine statische Liste mit Beispiel-Todos.

## Weitere Skripte

```bash
npm run lint
npm run build
npm run preview
```
